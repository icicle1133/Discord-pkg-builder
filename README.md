A simple and easy to use .pkg.tar.zst builder for Discord, since sometimes the pacman/aur repos aren't up-to-date. 

# Usage

1. run ```chmod +x install-deps.sh && chmod +X build.sh``` in the directory you put the files
2. run ```./install-deps.sh && ./build.sh``` < IN THAT EXACT ORDER (just in case you don't have the required dependencies installed
3. enjoy having discord installed
